# GLP Maritime - Site Web du Projet

## Description

Site web de présentation du projet **GLP Maritime**, un jeu vidéo de gestion et de stratégie navale développé dans le cadre de l'UE Génie Logiciel et Projet.

## Structure du Site

Le site est composé de 5 pages HTML :

1. **index.html** - Page d'accueil avec présentation générale du projet
2. **projet.html** - Spécifications détaillées et notions de base du jeu
3. **equipe.html** - Présentation de l'équipe de développement
4. **fonctionnalites.html** - Description des fonctionnalités du jeu
5. **contact.html** - Informations de contact

## Technologies Utilisées

- **HTML5** - Structure des pages (conforme W3C)
- **CSS3** - Mise en forme et design
- **Format XHTML5** - Pages validées XML

## Caractéristiques Techniques

- Pages HTML5 valides selon les validateurs W3C
- Utilisation des balises sémantiques : `header`, `main`, `nav`, `section`, `article`, `aside`, `footer`
- Navigation horizontale avec liste `<ul>` sans puces
- Fil d'Ariane sur les pages internes
- Design responsive adapté aux différents écrans
- Fichier CSS unique externe
- Métadonnées complètes (description, auteur, date, lieu)

## Contenu du Projet GLP Maritime

Le jeu GLP Maritime propose :

- **Gestion économique** : Ports maritimes, routes commerciales, ressources
- **Combat naval** : Flottes de bataille, conquête de villes, affrontements contre les pirates
- **Progression** : Système de trophées, déblocage de nouvelles villes et équipements
- **Économie complexe** : Impôts, équipements améliorables, navires de différents niveaux
